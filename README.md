# HomèVo Prototype

Modern responsive React + Tailwind prototype for ecological, sustainable and climate-adapted housing design.

## Run locally

```bash
npm install
npm run dev
```

Then open the Vite URL shown in the terminal.

## Structure

- `src/main.jsx` contains the page components, navigation and interactive UI.
- `src/prototypeData.jsx` contains housing models, material records, case studies and recommendation data.
- `src/locales/fr.js` and `src/locales/en.js` contain the bilingual text content used by React state.
- `src/styles.css` contains the Tailwind layers plus custom architectural visuals and map styling.

## Pages included

- Home
- Environmental Analysis
- Housing Models Library
- Materials Database
- Sustainable Construction Methods
- Professional Dashboard
- FR / EN language switcher with French as the default language
